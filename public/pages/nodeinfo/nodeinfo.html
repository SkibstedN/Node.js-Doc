Node.js info.

Node.js is a powerful runtime environment that enables developers to build high-performance applications using JavaScript. It is based on the V8 JavaScript engine and allows developers to use JavaScript on both the front-end and back-end of their applications. Node.js is known for its scalability and is often used for building web applications, APIs, real-time applications, and command-line tools.

Why Node.js?
Node.js uses asynchronous programming! Here is how Node.js handles a file request:
1. Sends the task to the computer's file system.
2. Ready to handle the next request.
3. When the file system has opened and read the file,
the server returns the content to the client.

Node.js eliminates the waiting, and simply continues with the next request. Node.js runs single-threaded, non-blocking, asynchronous programming, which is very memory efficient.

One of the key benefits of Node.js is its ability to handle large numbers of concurrent connections with low overhead. This makes it well-suited for building real-time web applications, such as chat applications or online multiplayer games.

What Can Node.js Do?
Node.js can generate dynamic page content.
Node.js can create, open, read, write, delete, and close files on the server.
Node.js can collect form data.
Node.js can add, delete, modify data in your database.

How?
To install Node.js, follow these steps:
1. Go to the Node.js website at https://nodejs.org/en/download/.
2. Download the version of Node.js that is appropriate for your operating system. Node.js supports a variety of platforms, including Windows, macOS, and Linux.
3. Run the installer and follow the instructions provided. The installation process will vary depending on your operating system.
4. Once the installation is complete, open a terminal or command prompt and type node -v to check the version of Node.js that was installed. If everything is working properly, you should see the version number printed to the screen.

That's it! You now have Node.js installed on your system and can start using it to build web applications.

In the context of our pensum we have been running Node.js in Visual Studio Code. Starting a Node.js project in Visual Studio Code involves several steps, including creating a new project directory, initializing a new Node.js project, and configuring Visual Studio Code to work with the project. Here are the basic steps of a way to do it:
Open a terminal window and navigate to the directory where you want to create your project. Use the mkdir command to create a new directory for your project, for example:

$ mkdir my-node-project

Initialize a new Node.js project: In the terminal window, navigate to the new project directory and use the npm init command to initialize a new Node.js project.

$ cd my-node-project npm init

This command will create a package.json file, which is used to define the metadata of the project, including its name, version, description, dependencies, scripts, and other important information. The file is also used by npm to manage the project's dependencies and allow other developers to install and use the project.

Use the npm install command to install any dependencies your project requires, for example:

$ npm install express

Create a new file:
In Visual Studio Code, create a new file in the project directory by selecting File > New File from the menu or using the keyboard shortcut Ctrl+N.
Save the file with a .js extension, for example, app.js.
In the new file, write Node.js code, for example:

                
                const express = require('express');
                const app = express();
                
                app.get('/', (req, res) => { 
                            res.send('Hello, world!');
                });
                
                app.listen(3000, () => { 
                    console.log('Server started on port 3000'); 
                });

This is a simple Node.js web server built using the Express.js framework that listens for HTTP requests on port 3000 and responds with the message "Hello, world!" when the root URL is accessed. Express will be explained in another document.

In the terminal window, use the node command to run your Node.js application. example:

$ node app.js

We have, however, installed the great nodemon tool to assist us in the development process so we will run the app with nodemon like this:

$ nodemon.cmd app.js

Nodemon will attempt to restart the app everytime a change in the sourcecode is made.
Nodemon will be further explained in another document.

The Node.js application should now be running, and can be viewed it in a web browser by navigating to http://localhost:3000/ .
